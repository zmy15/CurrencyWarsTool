# CurrencyWarsTool

`CurrencyWarsTool` 是一个基于 Avalonia UI 的崩坏：星穹铁道货币战争羁绊计算器桌面应用。

## 运行环境

- .NET 9 SDK

## 构建

```bash
dotnet build
```

## 运行

```bash
dotnet run
```
或下载Releases中发布的版本

## 项目功能

- 提供主界面面板布局，包含左右侧列表/网格区域与中间操作区。
- 中间操作区提供顶部与底部格子区域，可进行拖拽放置交互。
- 提供图片区域用于展示操作结果或信息。

## 项目结构

- `Views/` - Avalonia UI 视图。
- `ViewModels/` - 视图模型与界面逻辑。
